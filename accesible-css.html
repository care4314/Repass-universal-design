Overview

This page demonstrates an accessible multicolumn layout built with CSS Grid and Flexbox. Grid handles the two-dimensional structure—header, nav, main, asides, and footer—while Flexbox arranges the one-dimensional navigation bar.

Responsive Design

Media queries reflow the layout for tablets and mobile devices. On small screens, the nav switches to a vertical list with large, thumb-friendly targets.

Accessibility Features

  • Color-blind-safe, high-contrast palette.
  • Semantic HTML5 regions (header, nav, main, aside, footer).
  • Visible focus outlines for keyboard users.
  • Readable line length and font pairing from Google Fonts.

Aside A — Grid

CSS Grid defines both rows and columns, making it ideal for full page layout that needs structure in two dimensions.

Aside B — Flexbox

Flexbox arranges items along a single axis. Here it’s used for the navigation links so they stay evenly spaced and wrap gracefully.
